# BusRadar Product Specification v0.1

BusRadar è una webapp mobile-first standalone per visualizzare autobus urbani su mappa, con esperienza ispirata a un "FlightRadar dei bus".

Il progetto è separato da Turni Smart.

BusRadar v0.1 è una demo realistica con dati simulati, senza feed reali, senza scraping e senza endpoint non autorizzati.

## Target MVP

BusRadar Mobile MVP:

- smartphone prioritario;
- interfaccia moderna dark;
- mappa fullscreen;
- mezzi simulati;
- linee;
- vetture;
- fermate;
- radar;
- dettaglio vettura;
- percorsi linea.

## Stack

- React
- Vite
- TypeScript
- Leaflet
- CSS custom
- dati JSON/locali

## Vincoli

- Nessun backend.
- Nessun dato reale.
- Nessuna API GTT/MaTO/5T.
- Nessuno scraping.
- Nessun codice, logo o asset InfomezziMap.
- Nessun riferimento che faccia sembrare BusRadar ufficiale GTT.

## Success criteria

- Su smartphone sembra una vera app.
- La home mappa è credibile.
- I marker bus si muovono.
- La navigazione è fluida.
- Il dettaglio vettura funziona.
- Il radar funziona con dati simulati.
- Il progetto è pronto per futura integrazione dati reali.
